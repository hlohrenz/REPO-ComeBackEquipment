# ComeBackEquipment

![ComeBackEquipment Preview](preview.gif)

## Overview
This mod prevents equippable items from being destroyed when a player dies in a death pit.

Instead of disappearing, the items bounce back out so players can recover their equipment after falling in.

Works with modded items as well!
Hauler vehicles can also be sent back to the truck with a separate config option.

## Who Needs The Mod
All clients need the mod installed for it to work correctly.

## Configuration Values
| Setting | Description | Default |
|---------|-------------|---------|
| `TeleportItemsToTruck` | Enable to teleport equipment back to truck instead of bouncing back. | `false` |
| `TeleportHaulersToTruck` | Enable to teleport hauler vehicles back to truck instead of letting them be destroyed in a death pit. | `false` |
